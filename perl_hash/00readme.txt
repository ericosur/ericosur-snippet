demo scripts to show how to create hash by various way:

1. hash-key-value.pl: create hash by assignment

2. load-data.pl: create hash by "key=value" pair

3. map-array-hash.pl: use map() to create hash

4. hash-grep.pl: create hash from birth.txt, and use grep() to query data

5. hash-qw-list.pl: create hash from qw list

6. array-fib.pl: demo to create fib series

7. hash-map-array.pl: demo map, array
