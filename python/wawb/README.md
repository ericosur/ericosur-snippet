# readme for wawb

## rule
* answer is a 4 digit number, not leading with 0, no duplicated digits
* if match a digit with correct position, get 1A
* if match a digit with incorrect position, get 1B

for example:
answer is 8192, and
    * first guess is 1357, get 1B
    * guess 2468, get 2B
    * guess 9035, get 1B
    * guess 7249, get 2B
    ...
