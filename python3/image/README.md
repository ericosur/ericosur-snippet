# README

- [rainbow.py](./rainbow.py) uses PIL function with HSV values to generate a
  rainbow gradient color image.


## imageio

```
pip install imageio
```

- maybe ffmpeg is installed in PATH
- reference: https://imageio.readthedocs.io/en/stable/user_guide/requests.html
- use ```make``` to remove all jpg files


## scripts

- iterfiles.py
  - use PathLib to iterate all sub folders under specified src dir
  - list some sub folders that has space in their path name
- rainbow.py
  - use PIL to generate a color gradient image
- readpbf.py
  - alter the content in pbf file (potplayer bookmark file)
