## qrcode related

### generation

- qrcode
- segno

```python -m pip install segno```

```python
import segno
qrcode = segno.make_qr("hello world")
qrcode.save("qrcode.png", scale=5)
```

### view

ImageMagick view: ```magick display qrcode.png```

### decode

- zbar-tools

```
apt intall zbar-tools

zbarimg -q qrcode.png
QR-Code:hello world
```
